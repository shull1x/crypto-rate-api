# 🌾 Crypto Rate API

🙌 Простая API-обертка для получения актуальных курсов криптовалют к USDT. Работает через публичное API Binance.

## Запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите сервер:
   ```bash
   uvicorn main:app --reload
   ```

## Использование
После запуска перейдите по адресу: `http://127.0.0.1:8000/price/BTC`
Бот вернет JSON: `{"symbol": "BTC", "price": 95000.0}`
